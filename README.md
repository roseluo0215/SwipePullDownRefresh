# SwipePullDownRefresh
上拉刷新下拉加载
> 自定义类SwipePullDownRefresh继承android系统得下拉刷新类SwipeRefreshLayout
> 由于系统得类只提供了下拉刷新没有提供上拉加载功能，所以继承系统得类来扩展功能
